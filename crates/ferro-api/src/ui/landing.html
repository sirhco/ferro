Ferro

Rust-powered content engine.

Admin → Log in, manage content, types, users. API Docs → Swagger UI for the REST surface. GraphiQL → GraphQL queries + mutations. OpenAPI JSON → Machine-readable spec.

Quick links:

  • POST /api/v1/auth/login — exchange email + password for a JWT.
  • GET /api/v1/events?token=<jwt> — Server-Sent Events live preview.
  • WS /graphql/ws — GraphQL subscriptions (auth via connection_init).
Need an admin account? Run ferro admin create-user --with-admin --email ….
